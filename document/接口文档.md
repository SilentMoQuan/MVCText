## Student

### 获取单个学生信息

| 接口详情 |                   |
| -------- | ----------------- |
| 地址     | /student/get/{id} |
| 请求方式 | GET               |

| 参数 | 是否必填 | 说明                      |
| ---- | -------- | ------------------------- |
| {id} | 是       | url 参数 填写学生唯一学号 |

| 返回结果 | 格式       | JSON                                                         |
| -------- | :--------- | ------------------------------------------------------------ |
| 查询成功 | 普通返回体 | "status":200,<br />"message":null,<br />"data":{[学生实体信息]} |
| 查询失败 | 普通返回体 | "status":500,<br />"message":null,<br />"data":null          |



### 按照条件筛选学生信息

| 接口详情 |              |
| -------- | ------------ |
| 地址     | /student/get |
| 请求方式 | POST         |

| 参数                                                         | 是否必填 | 说明                                   |
| ------------------------------------------------------------ | -------- | -------------------------------------- |
| "info":{<br />"id":0,<br />"schoolName":"",<br />"gradeName":"",<br />"className":"",<br />"classroomRealId":"",<br />"realId":""<br />"name":""<br />} | 是       | 根据参数中提供的信息进行学生信息的筛选 |

| 返回结果 | 格式       | JSON                                                         |
| -------- | :--------- | ------------------------------------------------------------ |
| 查询成功 | 普通返回体 | "status":200,<br />"message":null,<br />"data":{[学生实体信息]} |
| 查询失败 | 普通返回体 | "status":500,<br />"message":null,<br />"data":null          |



### 添加学生信息

| 接口详情 |         |
| -------- | ------- |
| 地址     | /insert |
| 请求方式 | POST    |

| 参数                                                         | 是否必填 | 说明                   |
| ------------------------------------------------------------ | -------- | ---------------------- |

### 删除单个学生信息

| 接口详情 |                      |
| -------- | -------------------- |
| 地址     | /student/delete/{id} |
| 请求方式 | DELETE               |

| 参数 | 是否必填 | 说明                      |
| ---- | -------- | ------------------------- |
| {id} | 是       | url 参数 填写学生唯一学号 |

| 返回结果             | 格式       | JSON                                                         |
| -------------------- | :--------- | ------------------------------------------------------------ |
| 删除成功             | 普通返回体 | "status":200,<br />"message":null,<br />"data":null          |
| 删除失败             | 普通返回体 | "status":500,<br />"message":"学生信息删除失败",<br />"data":null |

### 删除多个学生信息

| 接口详情 |                 |
| -------- | --------------- |

| 请求方式 | DELETE          |

| 参数                     | 是否必填 | 说明                     |

| 返回结果             | 格式       | JSON                                                         |

| 删除成功             | 普通返回体 | "status":200,<br />"message":null,<br />"data":null          |
| 删除失败             | 普通返回体 | "status":500,<br />"message":"学生信息删除失败",<br />"data":null |
| 学生教师关联删除失败 | 普通返回体 | "status":500,<br />"message":"学生关联删除失败",<br />"data":{[发生错误的学生信息列表]} |



### 更新多个学生信息

| 接口详情 |                 |
| -------- | --------------- |
| 地址     | /student/update |
| 请求方式 | PUT             |

| 参数                                                         | 是否必填 | 说明                       |
| ------------------------------------------------------------ | -------- | -------------------------- |
| {<br />"info":{<br />"id":0,<br />"schoolName":"",<br />"gradeName":"",<br />"className":"",<br />"classroomRealId":"",<br />"realId":""<br />"name":""<br /><br />},<br />"idList":[1, 2, 3, 4,.....]<br />} | 是       | 根据信息完成学生数据的更新 |

| 学生教师关联删除失败 | 普通返回体 | "status":500,<br />"message":"学生关联删除失败",<br />"data":{[发生错误的学生信息列表]} |

| 学校班级不存在       | 普通返回体 | "status":500,<br />"message":"xxx学校xxx年级xxx班不存在, 请检查",<br />"data":null |
| 学生信息异常         | 普通返回体 | "status":500,<br />"message":"学生信息异常"",<br />"data":null |
